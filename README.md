## Overview

A mod that extends the baking process.

**dependent mods:** [Decorative Food](https://modrinth.com/mod/decorative-food).Don't worry,
This is a small mod and does not force new rules to be added to the game.

So it doesn't add an extra burden to the game

## Features

* Inheriting the mod's decorative food properties, almost all newly added foods can be placed.
* No complex GUI interactions, providing a more immersive baking experience.

## Approximate content

* A new stove! that is no longer a traditional single square, but a multi-square combination with the freedom to customize the appearance.
* New decorative blocks! Some kitchen utensils can be used to place a variety of items.
* New flour system,The raw materials are processed into dust using grinding stone, which can then be stored in dust flour sack.
* More bread! Using the stove, different doughs can be baked into corresponding breads.Detailed tutorial about the mod,Check out the wiki
